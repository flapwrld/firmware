This is a placeholder for the application icon.
To create a proper icon for your executable:

1. Create a 256x256 pixel PNG image
2. Convert it to .ico format for Windows executables
3. Name it "icon.ico" and place it in the project directory
4. Update the esp_controller_gui.spec file to reference it:
   icon='icon.ico'

For Linux, you can use the PNG file directly for desktop entries.
